Summary of the project:
The goal of the project is to learn more about the Unity game development platform, while improving the code abilities for game world physics. I enjoyed playing the game even more.

Game Instructions:
W- forward acceleration
A- Turn left
D- Turn right
The goal is to cross the road without getting stuck in an obstacle or falling out The more meters you pass, the faster the speed will be and the more points you will get!

Download Instructions:
To play the game click on play now and download the zip file from my github then extract the zip folder and enter the file called obslider.exe
