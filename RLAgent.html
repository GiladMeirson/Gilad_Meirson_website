Creating a smart agent using reinforcement learning to solve a maze

  • Gilad Meirson
  • #AI #Reinforcement-Learning

Summary of the project:
In order to learn the reinforcement learning algorithm, I built the following application of a smart agent that must find the cheese in the maze (the mouse).

In order to really learn how each parameter affects the agent, I created the configuration window on the left side so that the user can choose the value of each parameter himself to make the agent behave in a different way every run. It's a different style than a neural network but those who like artificial intelligence will enjoy this game project a lot. The algorithm is relatively simple.

Reinforcement learning (RL) is a type of machine learning algorithm where an agent learns to make decisions by interacting with an environment. The key components are:
1) Agent: The entity that learns and makes decisions
2) Environment: The world in which the agent operates
3) State: The current situation of the agent
4) Action: A decision made by the agent
5) Reward: Feedback from the environment on the action's quality


The algorithm works as follows:
* The agent observes the current state of the environment
* It chooses an action based on its policy
* The environment transitions to a new state
* The agent receives a reward
* The agent updates its policy to maximize future rewards

Over time, the agent learns to make better decisions by balancing exploration (trying new actions) and exploitation (using known good actions). Common RL algorithms include Q-learning, SARSA, and Policy Gradient methods.

Website Instructions:
To start playing the application, you must first select the values ​​for each parameter such as:
Alpha - the learning rate.
Gamma - discount rate on punishments or reinforcements.
Epsilon - the agent's rate of random decisions.
Episode - the option is given for the agent to start training before the start of the simulation, meaning that we can watch him being trained.
Grid Size - The size of the grid we want to play on.

First, click the Create Agent button.
If you didn't like the maze you can generate a new one.
In addition, you can view a graph that shows the agent's learning process.
