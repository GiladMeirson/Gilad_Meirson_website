Evolutionary AI-No Libraries

  • Gilad Meirson
  • #web #interactive #AI #NN
  • 4.85

Summary of the project:
I always had the idea of trying to code an AI that would know how to make decisions on its own, as we know the world of neural networks is evolving every moment, but I wanted to try to code myself an evolutionary AI whose goal is to survive like in nature. The results of the project are amazing and many conclusions can be drawn through this project.

What do we see?
Each circle on the screen represents an intelligent entity, Each run at the beginning decisions of the randomly drawn entities. After each iteration, the entities that finish the round inside the red area will die. The red area represents a dangerous area that cannot be lived or reproduced in, like simulating an area with high radiation or without a water source, etc.

Operating Instructions :
The first 4 inputs determine the size and location of the dangerous area. You can control the amount of population and the size of each circle (these parameters are important because the bigger the circle, the higher the chance of being in the dangerous area). You can determine how many runs for each population you want (EPOCS). After each end of all iterations you can view a graph that shows the state of the population in each iteration. On the left side you will see data for each iteration such as survival percentages, how many died, etc.
