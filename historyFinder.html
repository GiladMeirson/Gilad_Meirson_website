Summary of the project:
One of the easy but definitely fun projects I had the chance to work on.
Did you know that Wikipedia has an amazing API rich with information from which you can retrieve interesting events in human history?
Therefore, I decided to create a website that compiles these events.
Through user input of a day and month, I send a request to the Wikipedia API and retrieve the information, sorting it by years.
Additionally, there are images added from those events.

Website Instructions:
The instructions are quite simple.
All you need to do is enter a day and month, and you'll receive a list of interesting events that happened in human history, ordered by their occurrence.
Furthermore, when clicking on one of the events, you'll get more detailed information about what happened.
