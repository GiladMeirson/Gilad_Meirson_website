Creating a neural network + Mnist classification and digit generator

  • Gilad Meirson
  • #AI #Neural-network #brainJS
  • 4.9

Summary of the project:
One of the amazing projects I got to do.
It started with me creating a platform to create a neural network for users who don't know code but do know the principles behind basic neural networks.


After that, to check if the platform works properly, I wanted to build a neural network that manages to classify digits 0-9.
That is, the user enters a digit into the input and the network should recognize which digit he entered


Full disclosure - for the heavy algorithm I used a library called BrainJS which provides a variety of neural network functions such as building a network, training the network and prediction.

One thing led to another, and I had a crazy idea.
What would happen if we made the Mnist neuron network? (digit identification).
That is, instead of the user drawing a number and the network having to tell him what he drew, we will do the opposite, he will enter a number through input and the network will draw the number! That is, a generator of letters.
Admittedly in a simple way and without colors, but the principle behind it is not far from models that know how to create images for example. Another photo is attached below.

Website Instructions:
The instructions are quite simple, at first you have to determine the architecture of the model you are building.
After that you have to train the model exactly as in the template that appears in step 2 (in JSON format).
After the training phase you can download the model as a native "function" and use it for predictions.


The instructions for using the MNIST I built are quite simple:
In the test mode you have to draw a number and the model will tell you which number you drew and in what percentage he is sure of it.
In the generator mode you have to enter a digit as input and the amount of noise you want to insert and the model will draw the digit you entered!
